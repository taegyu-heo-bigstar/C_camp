# C_camp
영남대학교 ESC 동이리에서 진행하는 입문용 C언어 자료가 올라가는 repo입니다.

## 개요
본 캠프는 C언어 및 컴퓨터 공학에 대한 입문자를 대상으로 합니다.<br>
C 언어에 대한 기초적 개념을 배우고, 리눅스 환경에서 직접 실습해볼 예정입니다.<br>
주 1회 2시간씩 약 7주간의 일정이 예정되어 있으며, 자세한 것은 아래를 참고하시길 바랍니다.<br>

## 일정
본 캠프는 주 1회 2시간의 활동으로 약 7주간의 캠프 기간을 고려중에 있습니다.
- 1주차 : C와 리눅스가 무엇인지 알아보기, 실습환경 세팅하기
- 2주차 : 타입과 입출력 알아보기(with 아스키 코드, 유니 코드)
- 3주차 : 반복문과 활용
- 4주차 : 배열과 포인터, 문자열
- 5주차 : 함수와 터미널 환경 실습(with 객체지향)
- 6주차 : 헤더파일, 동적할당, 프로세스 이미지
- 7주차 : 간단한 자료구조 구현 실습(링크드 리스트 예정)

## 참고
실습된 모든 코드는 각각 개인의 repository에 업로드 할 예정입니다. 이를 위해 개인의 git hub 계정이 필요합니다.<br>
실습을 위해 리눅스 환경이 필요할 수 있습니다. 이를 위해 git hub codespace를 주로 활용할 예정이나,<br>
버추얼 박스나, wsl 등 보다 본격적인 환경을 마련할 수 있다면 이 쪽을 더 권장합니다.<br>
캠프 관련 자료는 각 주차 1주일 전 파일 혹은 링크로 배포될 예정입니다. 자세한 것은 채팅방을 확인해주세요,<br>
